# HelioFlux Web

Mobile-first scaffolding for the HelioFlux solar weather dashboard.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Cloudflare Worker Deployment

The application requires a Cloudflare Worker for API proxying. See [DEPLOY.md](DEPLOY.md) for quick deployment instructions.

- [DEPLOY.md](DEPLOY.md) - Quick deployment guide
